✅ How it works: Guests & RSVP data live in a Google Sheet. Any phone, tablet or laptop that opens this app reads from the same sheet in real time — no accounts, no sync buttons.
Google Sheets Connection
Paste your deployed Google Apps Script URL. See the one-time setup steps below.
📋 One-time Google Setup (5 min):
1. Open google-apps-script.gs → copy all the code
2. Go to script.google.com → New Project → paste code → Save
3. Click Deploy → New Deployment → Web App
4. Execute as: Me  ·  Who has access: Anyone
5. Click Deploy → Copy the Web App URL → paste above → Test
